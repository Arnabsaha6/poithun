# tg_music_bot
Getting Started
These instructions will get you a copy of the project up and running on your local machine.

Installing

use python 3.6.x  version

To use this bot, make a clone of this repository 
$ git clone https://github.com/manojpawarsj12/tg_music_bot.git

$ cd tg_music_bot

Install dependencies

$ pip install beautifulsoup4 python-telegram-bot requests youtube_dl

Get a bot token from @BotFather and replace YOUR-TOKEN in bot.py with the token you just got 

https://github.com/manojpawarsj12/tg_music_bot/blob/master/bot.py#L63

Run it:

$ python3 bot.py
